# Sistema de Gerenciamento –Clientes e Produtos

Este projeto é composto por duas partes:

- **Back-end (Node.js + Express + MySQL)**  
- **Front-end Mobile (Flutter + Provider+ HTTP)**  

A aplicação permite **cadastrar, listar, editar e excluir clientes e produtos**, integrando o flutter com uma API REST em Node.js.

---

##  Tecnologias Utilizadas

### **Back-end*
- Node.js
- Express
- MySQL
- mysql2
- CORS
- Nodemon

### **Front-end (Mobile)**
- Flutter
- Provider
- HTTP

---

## Requisitos

### **Back-end**
- Node.js instalado
- XAMPP ou MySQL Workbench
- Banco de dados MySQL criado:
  - Tabelas: `clientes` e `produtos`

### **Front-end**
- Flutter instalado
- Emulador Android ou celular físico
